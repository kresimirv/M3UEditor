M3UEditor is a lightweight editor for `.m3u` / `.m3u8` playlist files commonly used in IPTV streaming setups. It allows you to:

- Load and edit existing IPTV playlist files
- Add, remove, or modify channel entries
- Reorder channels for personalized view order
- Save modified playlists back to disk
- Keep your playlists tidy and compatible with media players

Supported file format is standard **M3U/M3U8** playlists.
